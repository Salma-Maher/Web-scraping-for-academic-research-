# Web-scraping-for-academic-research-
Web scraping for academic research 

A Python-based academic research assistant that automatically collects, ranks, and exports scientific papers from multiple scholarly sources. The project is designed to help researchers identify high-quality papers for literature reviews, surveys, and research projects.

Features,
Search academic papers by topic and publication year,
Retrieve paper metadata:
Title,
Authors,
Publication Year,
Citation Count,
DOI,
Abstract,
Venue (Conference/Journal),
Rank papers using a weighted scoring system,
Export results to Excel,
Identify top papers for survey studies,
Support recent publications (2020–2026),
Generate research-ready datasets,
